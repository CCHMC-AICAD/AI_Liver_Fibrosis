# AI_Liver_Fibrosis

Repository for **AI-based liver fibrosis analysis**, including model training, evaluation, inference utilities, and supporting code for imaging and multimodal experiments.

## Overview

This repository hosts the codebase and trained models used for liver fibrosis prediction and related analysis tasks. It is designed to support reproducible experiments across multiple feature types, including:

- **Radiomic features**
- **Deep learning image features**
- **Clinical features**
- **Multimodal / fused models**

The project can be adapted for binary or multi-class fibrosis classification, depending on label definition and dataset configuration.
